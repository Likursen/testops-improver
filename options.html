TestOps Improver

v3.0
Developed by DimaK
Focus Mode Включает скрытие бокового дерева тестов и панели с информацией при клике ЛКМ по иконке расширения.
Fix Copy (Cmd+C / Ctrl+C) Принудительное копирование текста из тест-кейса.
Jira Redirect Перенаправляет при клике ЛКМ по тест-кейсу в Jira на полные страницы в ТестОпс.
Smart Linker Превращает текстовые ключи задач в Jira в кликабельные ссылки.
Jira Project Prefix Префикс задач (например, ONECOLLECT).
