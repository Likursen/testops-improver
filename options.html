TestOps Improver

v6.3
Developed by DimaK
Focus Mode Клик по иконке расширения скрывает боковое дерево тестов и правую панель для расширения рабочей области.
Улучшенное копирование Исправляет проблему копирования текста в TestOps (используйте стандартные Ctrl+C / Cmd+C).
Jira Redirect Автоматическое открытие тест-кейсов в новой вкладке TestOps при клике в Jira (ЛКМ, колесико, Ctrl/Cmd+Клик).
Подсветка в дереве тестов Окрашивает активную строку мягким красным цветом и выделяет папки в дереве в которых расположен тест.
Smart Linker Превращает текстовые ключи задач (например, TASK-123) в кликабельные ссылки на Jira.
Префикс задач Jira Укажите буквенный код проекта (например, ONECOLLECT) для поиска ссылок.
Поиск кейса по ID Добавляет пункт в контекстное меню для перехода к тесту по выделенному номеру (например, #12345).
ID проекта TestOps Цифровой ID вашего проекта для корректной работы поиска и редиректов.
